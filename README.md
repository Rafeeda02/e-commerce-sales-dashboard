# E-Commerce Sales Dashboard

## Overview

This project presents an interactive Power BI dashboard to analyze e-commerce sales data. It helps in understanding sales performance, trends, and business insights.

## Tools Used

* Microsoft Excel – Data cleaning and preparation
* Microsoft Power BI – Dashboard creation and visualization

## Key Features

* Overall sales performance summary
* Sales by product and category
* Monthly sales trends
* Top-performing products
* Interactive filters for better analysis

## Data Description

The dataset includes:

* Product details
* Sales amount
* Quantity sold
* Order date

## Key Insights

* Certain product categories contribute the highest revenue
* Sales show variation across different months
* A small number of products generate most of the sales

## How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use filters/slicers to explore data
3. Review visuals to understand insights

## Purpose

This project demonstrates my ability to clean data, analyze it, and create meaningful dashboards using Power BI.
