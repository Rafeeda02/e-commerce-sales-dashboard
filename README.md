# E-Commerce Sales Dashboard

## Overview

This project presents an interactive Power BI dashboard built to analyze e-commerce sales data. It provides a clear view of sales performance, trends, and key business insights.

## Tools Used

* Microsoft Excel – Data cleaning and preparation
* Microsoft Power BI – Data visualization and dashboard creation

## Key Features

* Overall sales performance summary
* Sales analysis by product and category
* Time-based trends (monthly/yearly)
* Identification of top-performing products
* Interactive filters for better data exploration

## Data Description

The dataset includes:

* Product information
* Sales amount
* Quantity sold
* Order date

## Key Insights

* Highlights best-selling products
* Shows sales patterns over time
* Helps compare category performance

## How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use slicers/filters to explore data
3. Review charts and visuals for insights

## Purpose

This project is developed to demonstrate data analysis and visualization skills using Power BI, focusing on turning raw data into meaningful insights.
